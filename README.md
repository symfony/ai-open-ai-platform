OpenAI Platform
===============

Provides [OpenAI](https://openai.com/) platform integration for Symfony AI Platform.

This bridge includes support for:
- GPT models (chat completions)
- Embeddings
- DALL-E (image generation)
- Whisper (audio transcription)
- Text-to-Speech
